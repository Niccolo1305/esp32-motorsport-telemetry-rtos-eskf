# M5Stack AT6668 Evidence Pack v1.5.2

This folder contains the evidence package referenced by the whitepaper:

- `00_report`
  - `AT6668_DHV_NAVPV_Findings_v1.5.2.pdf`
  - `AT6668_DHV_NAVPV_Findings_v1.5.2.md`
- `01_artifacts`
  - `A1` to `A5`: offline analysis artifacts cited in the report
  - `M0` to `M3`: MQTT captures cited in the report
- `02_datasheets`
  - `R1` to `R3`: local datasheet references used during the investigation
- `03_public_sources`
  - `P1`: public M5Stack product PDF included locally
  - `P2` to `P4`: internet shortcuts plus a local index for public web references

Notes:

- This pack is scoped to the tested M5Stack / AT6668 module, tested firmware, and tested command path described in the report.
- Raw `.bin` logs are not included here because they were not directly cited in the vendor-facing whitepaper. They can be added separately if requested.
